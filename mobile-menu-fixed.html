Test Instructions

  1. Resize your browser window to mobile size (< 768px) or use mobile device
  2. Click the hamburger menu button (three lines) in the top right
  3. Verify the menu slides in from the right with SOLID WHITE background
  4. Test the Services submenu by clicking on it
  5. Try closing the menu using the X button or clicking the overlay
  6. Test that the hamburger animates to an X when open

Expected Behavior

  • SOLID WHITE BACKGROUND - No content bleeding through
  • Menu should slide in smoothly from the right
  • Overlay should appear behind the menu
  • Hamburger should animate to X when menu is open
  • Services submenu should expand/collapse when clicked
  • Menu should close when clicking X, overlay, or pressing Escape
